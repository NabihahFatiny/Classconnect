# ClassConnect – TEACHING AND LEARNING MANAGEMENT PORTAL

A web-based platform designed to help teachers and students manage lessons, assignments, and communication in one centralized system. The system improves organization, collaboration, and efficiency in the teaching and learning process. Teachers can upload lessons, create assignments, and provide feedback, while students can access materials, submit assignments, and participate in discussions.

## Features

• User registration and secure login (role-based access for teachers and students) • Lesson management (add, edit, delete, and view lessons) • Assignment management (create assignments, submit work, and provide feedback) • Discussion and communication system • User profile management • Centralized dashboard for learning activities
